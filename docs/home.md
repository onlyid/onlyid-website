# 唯ID 开发文档

:::tip 标语
我们坚信：**账号认证** 和言论自由、互联网安全一样，是人民的基本权利。
:::

欢迎使用唯ID 统一、专业、安全的账号和认证解决方案。

## 项目背景

一直以来，人们习惯在每个用到的应用都注册一个单独的账号。这导致了两个问题：

* 账号密码众多，难以记忆和管理，如果都使用同个账号密码，一旦密码泄露，会造成严重的安全问题。
* 小众应用更难吸引用户注册，头部应用（微信等）占据更多流量，互联网逐渐远离开放、自由的精神。

在此背景下，唯ID应运而生。

## 开始使用

我们倡导渐进式开发。

如果你刚刚接触唯ID，可以只使用OTP功能发送验证码，节省短信成本的同时也最容易使用。

在对唯ID稍了解后，就可以接入SSO，这是唯ID的统一认证中心。最大的好处是，用户只需要一个账号就可以登录所有接入的应用。

从<i className="iconfont docs__left-hand">&#xe78d;</i>左边目录挑选感兴趣的章节，马上开始吧。

:::info 提示
在调用唯ID的OpenAPI时，请求成功返回HTTP状态码200，请求失败返回40X（一般是400），然后通过error字段给出具体错误原因。
:::
